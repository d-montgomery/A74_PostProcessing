# Average Spray Plotting

The `make_plots_paraview_integrals.py` script relies on data exported from ParaView via the `trace_paraview_integrals.py` script. The data should be exported to directories with the following format: `data_paraview_integrals_<start-time>_<end-time>`.

The `trace_paraview_integrals.py` script can be used on any plot file, but it is best used on time-averaged data.
